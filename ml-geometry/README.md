# ml-geometry

> Machine learning operators for geometry processing (Python)

## Vision

Integration of neural operators with geometric processing:
- **FNO/GNO** - Fourier and Graph Neural Operators
- **G-CNN** - Group-equivariant convolutions for geometry
- **LLM Bridge** - Natural language to SST compilation
